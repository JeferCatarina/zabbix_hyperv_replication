Monitoramento replica Hyper-v com descoberta (LLD)

Requisitos: Zabbix Server 3.2+

1. Baixe os arquivos;
2. Importe o template;
3. Copie os scripts para o local de sua preferencia;
4. Ajuste o zabbix_agent.conf conforme exemplo anexo, realizando os ajustes necessários no caminho dos scripts;

Status na coleta hyperv_replica[health,SRVNAME]:
0 - OK
1 - Warning
2 - Critical
3 - Unknown
